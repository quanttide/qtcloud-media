# 小红书制作器

## 工作流程

```
原始内容 → LLM 生成结构化数据 → HTML 模板渲染 → 浏览器截图 → 小红书图片
```

## 步骤

### 1. 准备原始内容

- 主题：确定要发布的内容主题
- 素材：收集相关文字、数据、引用

### 2. LLM 生成结构化数据

输入提示词模板：

```
请把以下内容整理成小红书图片的结构化数据：

主题：{topic}
原始内容：{content}

输出 JSON 格式：
{
  "pages": [
    {
      "type": "cover",
      "title": "主标题",
      "subtitle": "副标题"
    },
    {
      "type": "content",
      "heading": "标题",
      "points": ["要点1", "要点2", "要点3"]
    }
  ]
}
```

### 3. HTML 模板渲染

- 使用预定义的 HTML 模板
- 根据结构化数据填充内容
- 生成完整的 HTML 文件

### 4. 浏览器截图

使用 playwright 或手动截图：

```bash
uv run python html2img.py
```

### 5. 发布到小红书

- 导出图片
- 调整尺寸和顺序
- 发布

## 模板类型

| 类型 | 用途 |
|------|------|
| 封面 | 吸引眼球的主图 |
| 对比 | 前后对比、优缺点 |
| 步骤 | 教程、流程 |
| 清单 | 要点列表 |
| 时间线 | 计划、历程 |
| 行动号召 | 引导用户行动 |
