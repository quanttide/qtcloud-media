Vibe Coding 新手入门指南

Vibe Coding（氛围编程）是一种利用 AI 辅助编程的新方式。与传统编程不同，它更关注「你想要什么效果」，而不是「怎么实现」。

核心理念： 用自然语言描述需求，AI 生成代码，你来审查和完善。

什么是 Vibe Coding？

传统编程：学会编程语言 → 自己写代码 → 调试修复

Vibe Coding：描述需求 → AI 生成代码 → 审查和调整

新手入门步骤

1
选择 AI 工具
  • Cursor - 最流行的 AI IDE
  • Windsurf - 注重体验的 AI 编辑器
  • GitHub Copilot - 集成在 VS Code 中
2
从简单项目开始
  • 不要一开始就想做复杂应用
  • 从一个 HTML 页面、一个脚本开始
  • 目标是跑通流程，不是做完美产品
3
学会描述需求

好的 Prompt 是成功的一半：

  • 明确告诉 AI 你要做什么
  • 说明目标用户和使用场景
  • 描述你期望的效果
4
学会审查代码
  • AI 生成的代码不一定正确
  • 你需要理解代码在做什么
  • 知道如何修改和调试

新手常见误区

误区一：完全交给 AI

AI 是工具，不是替代品。你仍然需要理解基本概念和逻辑。

误区二：一次性生成完整项目

建议分步骤：小功能 → 测试 → 调整 → 下一个功能。

误区三：不学习基础

Vibe Coding 不等于不需要学编程。基础概念还是要懂的。

推荐学习路径

  1. Week 1: 安装工具，体验 AI 补全功能
  2. Week 2: 用 AI 写一个简单网页
  3. Week 3: 尝试修改和完善 AI 生成的代码
  4. Week 4: 做一个完整的小项目
学习资源：
  • Cursor 官方文档
  • Learn Prompting - AI 提示词学习
  • Vibe Coding 社区

核心能力

  • 需求描述 - 清晰表达你想要什么
  • 代码审查 - 判断代码是否正确
  • 调试能力 - 发现并修复问题
  • 迭代优化 - 不断完善和改进

开始吧！

不需要等到「学会所有东西」再开始。Vibe Coding 的精髓就是「边做边学」。

找一个简单的想法，比如「帮我写一个 todo list」，然后开始吧！

本指南面向编程新人，帮助理解 Vibe Coding 的基本概念和学习路径。
